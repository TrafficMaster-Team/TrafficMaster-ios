# TrafficMaster

Приложение для изучения ПДД ГАИ РБ с использованием методики интервальных повторений (Anki) и геймификации.

## 📱 О проекте
Мы создаем инструмент, который поможет курсантам автошкол сдать экзамен в ГАИ с первого раза. В основе лежит умный алгоритм, который подстраивает частоту показа вопросов под ваши ошибки.


## 🛠 Технический стек
- **Язык:** Swift 6
- **UI:** SwiftUI
- **Архитектура:** MVVM
- **Минимальная версия:** iOS 17.0
- **Максимальная iOS версия:** iOS 26.3
- **Работа с сетью:** хз ещё)))
- **Локальное хранилище:** SwiftData

## 🚀 Функционал MVP
- [ ] Экран с карточками вопросов ПДД
- [ ] Алгоритм интервальных повторений (SM-2) (Интеграция с ним скорее для правильной высылки на бэк)
- [ ] AI-объяснения ошибок
- [ ] Синхронизация с Backend (FastAPI)
