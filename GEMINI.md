# TrafficMaster - Инструкции и Видение Проекта

Этот файл является основным руководством для разработки проекта TrafficMaster. Все изменения и архитектурные решения должны соответствовать описанным ниже принципам.

## 🛠 Технический стек и инструменты

### Xcode MCP (Обязательно к использованию)
Для взаимодействия с проектом **необходимо** использовать инструменты Xcode MCP. Это обеспечивает синхронизацию между кодом и структурой проекта Xcode.

**Основные юзкейсы Xcode MCP:**
- **XcodeLS**: Исследование структуры групп и файлов проекта (не путать с файловой системой).
- **XcodeRead / XcodeWrite**: Чтение и запись файлов с автоматическим обновлением `.pbxproj`.
- **XcodeUpdate**: Точечное редактирование содержимого файлов.
- **BuildProject**: Проверка компилируемости после изменений.
- **RunAllTests / RunSomeTests**: Валидация изменений через систему тестирования.
- **RenderPreview**: Проверка UI-компонентов SwiftUI через Swift Previews.
- **XcodeListNavigatorIssues**: Мониторинг ошибок компиляции и предупреждений в реальном времени.

### Архитектура
- **MVVM (Model-View-ViewModel)**: Обязательное разделение ответственности. 
    - **Model**: SwiftData модели (например, `Question.swift`).
    - **View**: SwiftUI представления, не содержащие бизнес-логики.
    - **ViewModel**: Классы с аннотацией `@Observable` для управления состоянием и логикой.

## 🚦 О проекте: TrafficMaster
Приложение для изучения ПДД ГАИ РБ посредством методики **Anki** (умная алгоритмическая шкала запоминания информации) с элементами геймификации.

### Платформы
- **iOS 17-26** (Основной приоритет)
- **Web** (Вторично, в долгосрочной перспективе)

## 🎯 Vision

### MVP (Ближайшие 4–8 недель)
- iOS-приложение с актуальными карточками по ПДД РБ.
- Система интервальных повторений (логика SM-2/Anki).
- **AI-ассистент**: Объяснения ошибок и контекстные подсказки на уровне конкретного вопроса.

### Future
- Персональные учебные планы на основе прогресса.
- Диагностика слабых тем и автоматическая генерация "умных" подборок.
- Полноценный режим "Экзамен" (таймер, симуляция условий ГАИ, детальная статистика).

## 👥 User Personas
1. **Студент автошколы**: Хочет сдать теорию с первого раза. Нуждается в понятных объяснениях, а не просто заучивании.
2. **«Лишенец»**: Нужно быстро и эффективно восстановить знания в условиях стресса. Требуется фокус на проблемных зонах.
3. **Опытный водитель**: Хочет освежить знания, чтобы избегать штрафов. Нужны короткие, эффективные сессии.
