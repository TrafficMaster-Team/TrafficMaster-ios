# 🐛 Исправление дублирования нумерации в вариантах ответов

## ❌ Проблема

**Было:**
```
Вопрос: "Что означает красный сигнал светофора?"

Варианты:
┌───┬─────────────────────────────────────┐
│ 1 │ 1. Запрещает движение               │  ← ДУБЛИРОВАНИЕ!
├───┼─────────────────────────────────────┤
│ 2 │ 2. Разрешает движение               │  ← ДУБЛИРОВАНИЕ!
├───┼─────────────────────────────────────┤
│ 3 │ 3. Предупреждает об опасности       │  ← ДУБЛИРОВАНИЕ!
└───┴─────────────────────────────────────┘
```

**Проблема:** В базе данных варианты уже содержат нумерацию ("1. Запрещает..."), а приложение добавляет свою → получается "1. 1. Запрещает..."

---

## ✅ Решение

### 1. Добавлен метод очистки `cleanOptionText()`

**Файл:** `Question.swift`

```swift
/// Очищает вариант ответа от нумерации в начале
func cleanOptionText(_ option: String) -> String {
    // Удаляем нумерацию в форматах:
    // "1. Текст" → "Текст"
    // "2) Текст" → "Текст"
    // "3)Текст" → "Текст"
    let pattern = #"^\s*\d+[\.\)\s]\s*"#
    return option.replacingOccurrences(of: pattern, with: "", options: .regularExpression)
}
```

**Работает для форматов:**
- `"1. Текст"` → `"Текст"`
- `"2) Текст"` → `"Текст"`
- `"3)Текст"` → `"Текст"`
- `"  4.  Текст"` → `"Текст"` (с пробелами)

---

### 2. Обновлён рендеринг вариантов

**Файл:** `QuestionView.swift`

**Было:**
```swift
Text(option)  // Показывает "1. Запрещает движение"
```

**Стало:**
```swift
// Нумерация варианта (1, 2, 3, 4)
Text("\(index + 1)")

// Текст варианта БЕЗ встроенной нумерации
Text(viewModel.currentQuestion?.cleanOptionText(option) ?? option)
```

## 📊 Дополнительные исправления для марафона

### Исправлен баг с "нет вопросов"

**Файл:** `QuestionViewModel.swift`

**Проблема:** Если ID вопроса в марафоне не найден в базе → марафон завершался с ошибкой.

**Решение:**
```swift
private func loadNextMarathonQuestion() {
    // Skip any IDs that don't have matching questions
    while !marathonRemainingIds.isEmpty {
        let nextId = marathonRemainingIds.first!
        
        if let question = allQuestions.first(where: { $0.id == nextId }) {
            currentQuestion = question
            shuffledOptions = question.options.shuffled()
            saveMarathonProgress()
            return
        } else {
            // Question not found - remove this ID and try next
            print("⚠️ Marathon: Question with ID \(nextId) not found, skipping...")
            marathonRemainingIds.removeFirst()
        }
    }
    
    currentQuestion = nil
}
```

**Теперь:**
- ✅ Пропускает несуществующие вопросы
- ✅ Логгирует проблему
- ✅ Продолжает марафон
