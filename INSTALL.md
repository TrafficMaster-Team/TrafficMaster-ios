# 📱 TrafficMaster - Инструкция по установке на iPhone

## ✅ Требования
- iPhone с iOS 17.0 или новее
- Xcode на Mac (последняя версия)
- Apple ID (для подписи приложения)

---

## 🔧 Шаг 1: Откройте проект в Xcode

```bash
cd /Users/vlad/TM/TrafficMaster
open TrafficMaster.xcodeproj
```

Или просто откройте файл `TrafficMaster.xcodeproj` через Xcode.

---

## 🔧 Шаг 2: Настройте подписывание

1. В Xcode выберите проект **TrafficMaster** в навигаторе слева
2. Выберите таргет **TrafficMaster** (иконка приложения)
3. Перейдите на вкладку **Signing & Capabilities**
4. В секции **Team** выберите:
   - Ваш Apple ID (если нет — добавьте в Xcode → Settings → Accounts)
   - Или создайте новую команду

**Важно:** Убедитесь, что **Bundle Identifier** уникален:
```
OrDinaD.TrafficMaster
```

Если ошибка "Bundle identifier already taken", измените на:
```
com.yourname.TrafficMaster
```

---

## 🔧 Шаг 3: Выберите устройство

1. В верхней панели Xcode нажмите на название устройства (например, "iPhone 17 Pro")
2. Выберите ваше реальное устройство из списка
3. Если устройство не отображается:
   - Подключите iPhone кабелем
   - Разблокируйте iPhone
   - На iPhone: Настройки → Основные → VPN и управление устройством → Доверять этому компьютеру

---

## 🔧 Шаг 4: Установите приложение

1. Нажмите **▶️ Run** (или `Cmd + R`)
2. На iPhone появится запрос "Доверять разработчику?"
   - Нажмите **Доверять**
3. Приложение установится и запустится

---

## ⚠️ Решение проблем

### ❌ "Untrusted Developer"

Если приложение не запускается:

1. iPhone: **Настройки** → **Основные** → **VPN и управление устройством**
2. Найдите ваш Apple ID в разделе "Разработчик"
3. Нажмите **Доверять** → **Доверять** ещё раз

### ❌ "Code signing failed"

Проверьте:
1. В Xcode: **File** → **Project Settings** → **Signing**
2. Убедитесь, что **Automatically manage signing** включено
3. Попробуйте сменить Team на **Personal Team**

### ❌ "No provisioning profiles found"

1. Xcode: **Xcode** → **Settings** → **Accounts**
2. Выберите ваш Apple ID
3. Нажмите **Manage Certificates**
4. Нажмите **+** → **iOS Development**
5. Закройте окно и попробуйте снова

### ❌ Приложение вылетает при запуске

Проверьте логи:
1. Xcode: **View** → **Navigators** → **Show Issue Navigator** (`Cmd + 4`)
2. Или: **Window** → **Devices and Simulators**

Возможные причины:
- Не загрузились данные (проверьте консоль на "Starting background data import")
- Проблема с JSON (проверьте файл `adrive_questions.json`)

## 📊 Проверка установки

После успешного запуска вы должны увидеть:

1. Экран загрузки "Загрузка данных..."
2. Через 5-30 секунд (после импорта 3000 вопросов) — экран входа
3. Введите имя → Профиль → Главная страница с путём обучения

---

## 🔄 Обновление приложения

При изменении кода:

1. Просто нажмите **▶️ Run** снова
2. Приложение перезапишется на устройстве
3. Данные сохранятся (SwiftData персистентна)

**Сброс данных** (если нужно):
1. Удалите приложение с iPhone (долгое нажатие → Удалить)
2. Установите заново

---

## 📱 Установка на другие устройства

Чтобы установить на другой iPhone/iPad:

1. Подключите устройство к Mac
2. Выберите его в Xcode
3. Нажмите **Run**

Или используйте **TestFlight** (требуется Apple Developer Program за $99/год).
